Dyna Blaster

This package contains (limited) Dyna Blaster reimplementation in Java. We strived for consistency with original Dyna, although certain things might work a bit different than in the original.

The reimplementation is limited to tournament (battle) mode only. No single-player mode is implemented.

Note that copyrights for the graphical files and resources may (and probably still do) hold, so this code is for internal use only, please do not publish or redistribute.
